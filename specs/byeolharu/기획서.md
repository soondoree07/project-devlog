## 0. v1.1 변경 요약

후회 별에 더해 **성취 별**(잘한 일·해낸 일·칭찬받고 싶은 일)을 추가. **"밝은 별 = 성취"** 정의를 통일하고, 밝은 별에 도달하는 경로를 두 가지로 확장:

- **후회 극복 경로**: 흐린 별 → 다짐·행동 → 밝은 별
- **성취 직접 입력 경로**: 입력 순간 바로 밝은 별

하늘·슬롯·6 카테고리는 그대로 공유. 영향 섹션: 1, 3.3, 6, 7.1, 7.2, 7.3, 7.5.

## 1. 제품 요약

**한 줄 요약:** 후회를 성장의 연료로 변환하고, 성취를 별로 남기는 심리학 기반 개인 성장 모바일 앱.

**핵심 가치 제안:** "후회를 변환하는 가장 아름다운 1분, 나만의 밤하늘을 만들어간다."

유저는 자신의 후회를 기록하고 다짐으로 전환한 뒤 극복 과정을 별과 별자리로 경험한다. 더해 자신이 잘한 일·해낸 일을 **성취 별로 직접** 기록할 수도 있다. **밝은 별 = 성취**라는 하나의 정의 아래 두 경로 모두 밝은 별로 모이며, 이 별들이 모여 **나만의 밤하늘**을 완성한다.

### 제품 철학 세 축

1. **Transform, not Record** — 기록보다 변환. 일기 앱이 아님
2. **Regret-Positive Framing** — 후회는 "더 나은 선택지를 안다는 증거"
3. **밝은 별 = 성취 (두 갈래의 한 목적지)** — 후회 극복도 결국 하나의 성취

## 3. 핵심 기능

### 3.1 새 별 만들기 진입 — 후회 / 성취 분기 (v1.1 신규)

```
[새 별 만들기 (+)]
  ↓
[후회 / 성취 선택]
  ├─ 후회 → 3단계 위저드
  └─ 성취 → 2단계 위저드
```

### 3.2 후회 기록 플로우 (3단계, 1분 이내)

| 단계 | 액션 | 설명 |
|---|---|---|
| 1 | 카테고리 선택 | 6개 중 택1 |
| 2 | 후회 한 줄 입력 | 희미한 별 생성 애니메이션 |
| 3 | 다짐 + 리뷰 날짜 + 별 이름(선택, 1~8자) | "또 같은 일이 일어나면 어떻게 할래?" |

### 3.3 성취 기록 플로우 (2단계, v1.1 신규)

| 단계 | 액션 | 설명 |
|---|---|---|
| 1 | 카테고리 선택 | 6개 중 택1 (후회와 공유) |
| 2 | 성취 한 줄 입력 + 별 이름(선택, 1~8자) | **즉시 밝은 별 생성 애니메이션** |

**설계 원칙:**
- 이미 해낸 일이므로 **다짐·리뷰 날짜 단계 없음**
- 흐림 단계 없이 입력 순간 바로 밝은 별로 슬롯 점유·완성
- 가볍고 부담 없는 2단계로 진입 장벽 최소화

**공통:** 감정 기록 단계 의도적 생략. 각 플로우 1분 이내 목표.

### 3.4 극복 선언 플로우 (2단계 분리, 후회 별에만 적용)

```
[희미한 별 탭]
  ↓ [극복했어요]
[✨ 풀컬러 모핑 + 효과음 + 햅틱]  ← 즉시 보상 (0.8~1.2초)
  ↓ (1~2초 여운)
[부드러운 입력창 페이드인]
  → "이 극복의 순간을 기록해볼까요?"
  ├─ 스토리 입력 → 저장 + 아카이브
  └─ "나중에 할게요" → 저장 (스토리 미작성, 빛나는 별 재탭 시 1회 재프롬프트)
```

**설계 근거:** 변환 경험이 입력 장벽보다 우선. 몰입 최고조 유지.

### 3.5 리뷰 날짜 도래 (후회 별만 해당)

푸시 알림 → 앱 진입 → 해당 별 하이라이트 → 3개 선택지:
- **재도전** (리뷰 날짜 재설정)
- **내려놓기** (별 시들어 사라짐, 슬롯 해제)
- **그대로 두기** (상태 유지)

### 3.6 별자리 시스템

**진행 구조 (고정):** 북두칠성 → 큰곰자리 → 작은곰자리 → 자유 해금 (무료: 랜덤 / 프리미엄: 선택·생성 / 생일 별자리 옵션)

**완성 조건:** 별자리 모든 슬롯이 풀컬러로 채워졌을 때 (후회 극복 별과 성취 별을 구분하지 않음).

**상태 전이:**
- 후회 별 생성 → 슬롯 점유 (희미한 상태)
- **성취 별 생성 → 슬롯 점유 + 즉시 완성 (풀컬러)** — 후회를 거치지 않고도 별자리를 채울 수 있음
- 내려놓기 → 슬롯 해제 → 가장 오래된 플로팅 별 자동 편입
- 재도전 → 슬롯 유지 (희미한 상태 지속)

**북두칠성 (슬롯 7개 고정):**

| 슬롯 | 한자 | 로마자 |
|---|---|---|
| 1 | 천추(天樞) | Dubhe |
| 2 | 천선(天璇) | Merak |
| 3 | 천기(天璣) | Phecda |
| 4 | 천권(天權) | Megrez |
| 5 | 옥형(玉衡) | Alioth |
| 6 | 개양(開陽) | Mizar |
| 7 | 요광(搖光) | Alkaid |

- 슬롯 이름 기본: 유저 이름 또는 AI 추천어 / 정보 아이콘 탭 시 원래 이름·의미 표시
- 북두칠성 완성 시 커뮤니티 인증 메시지 ("당신은 N번째 별지기입니다")

**플로팅 별:**
- 슬롯 모두 점유 시 새 별(후회·성취 모두)은 플로팅으로 생성 (성취는 밝은 플로팅 별)
- 슬롯 해제 시 가장 오래된 플로팅 별 자동 편입
- 3개 이상 누적 시 소프트 넛지 ("떠 있는 별이 3개 있어요. 혹시 다시 볼 시간이 됐을까요?")

### 3.8 별자리 아카이브 & 히스토리 타임머신

- **아카이브(무료)**: 별자리 완성 시 자동 저장. 별 정보에 종류(후회/성취), 원본 내용, 극복 스토리(후회 별만), 카테고리, 날짜 포함
- **별하루 세상**(커뮤니티 현황): 총 별지기 수, 빛나는 별 총계, 완성된 별자리 수
- **히스토리 타임머신(v1.5 프리미엄)**: 완성된 별자리를 영화처럼 자동 재생

### 3.9 별 탭 인터랙션

**희미한 별 탭 (후회 별만):**
- 원본 후회 + 다짐 + [극복했어요] 버튼
- 리뷰 날짜 도래 시 [재도전][내려놓기][그대로 두기] 추가

**빛나는 별 탭:**
- (후회 극복 별) 원본 후회 + 극복 스토리 + 생성/극복 날짜 + 걸린 기간
- (성취 별) 원본 성취 내용 + 생성 날짜 (다짐·극복 단계 없음)
- 카테고리 배지
