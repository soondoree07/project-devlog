# Flow Project 기획서 (v0.4)

> 시간에 따라 변하는 비율을 시각적으로 보여주는 데이터 시리즈 사이트
>
> **사이트 이름은 가칭 "Flow Project"이며, 추후 정식 명명 예정.**

---

## 1. 프로젝트 개요

### 1-1. 한 줄 소개
"세상이 어떻게 바뀌어 왔는지를 도형의 비율로 보여주는 데이터 시각화 사이트"

### 1-2. 컨셉의 출발점
- 송스패치 채널의 "에스파 영토 시뮬레이션" 영상이 모티브
- 사각형(또는 다른 도형) 안의 영역 비율이 시간에 따라 부드럽게 변하는 형태
- 같은 시각화 문법을 다양한 주제에 적용해 "시리즈"로 운영

### 1-3. 차별화 포인트
1. **레퍼런스와의 차별**: 송스패치는 한 영상 = 한 주제. 본 프로젝트는 사이트 = 5개 이상 시리즈의 허브
2. **도형의 다양화**: 사각형 외에 보로노이/벌집/서클패킹/스트림그래프 등을 주제별로 다르게 적용
3. **인터랙티브성**: 영상은 일방향. 사이트는 사용자가 특정 시점을 직접 탐색 가능
4. **내러티브 캡션**: 단순 차트가 아니라 "왜 이 시점에 이렇게 변했는지" 이벤트 주석을 함께 제공

### 1-4. 작업 단위 명칭
- 영문: **Stories**
- 한글: **이야기** 또는 **시리즈**
- URL 패턴: `/stories/{series-slug}` (예: `/stories/pokemon-types`)

### 1-5. 사이트 이름 / 도메인 (가칭)
- **현재 가칭: Flow Project**
- 추후 정식 이름 결정 예정 (검토 중인 후보: Eraflow, Gyeol(결), Saecula, Neoul(너울) 등 — "시대의 흐름" 의미 담긴 신조어 우선)
- 도메인 후보 미정 (서브도메인 활용 가능: `flow.junghyeok.com` 등)

## 2. 핵심 시각화 문법

### 2-1. 공통 구조
모든 시리즈는 다음 공통 요소를 가진다.

| 요소 | 설명 |
|---|---|
| **시간 축** | 연도 / 세대 / 패치 / 시즌 등 시리즈마다 다름 |
| **영역(도형)** | 카테고리별 영역. 비율에 비례하는 크기 |
| **색상 시스템** | 카테고리별 고유 색. 한 번 정해지면 끝까지 유지 |
| **타임 인디케이터** | 현재 시점 표시 (연도, 세대 등) |
| **이벤트 캡션** | 해당 시점의 주요 사건 텍스트 |
| **재생 컨트롤** | 재생 / 일시정지 / 속도 / 시점 이동 |

### 2-3. 인터랙션 모드
사용자는 두 가지 모드를 토글할 수 있다.

- **자동 재생 모드**: 알아서 흘러감. 일시정지/속도 조절. 모바일 기본값
- **직접 탐색 모드**: 슬라이더로 시점 이동. 데스크탑 기본값
- **공유 기능**: 특정 시점의 URL 공유 (`?t=2020`)

### 2-4. 공용 인터랙션 폴리시 (v0.6)

모든 시리즈에 공통으로 적용되는 인터랙션·애니메이션 요소.

| 기능 | 설명 | 상태 |
|---|---|---|
| **숫자 카운터 애니메이션** | 비율 12.4% → 14.7% 변할 때 rAF로 부드럽게 보간 (CountUp.js 스타일) | ✅ |
| **순위 변동 화살표** | ↑N 녹색 / ↓N 빨강 / NEW 뱃지 | ✅ |
| **랭킹 사이드바** | 도형 외 우측에 TOP 5 텍스트 리스트 (영역+텍스트 동시 노출) | ✅ |
| **연도/세대 슬롯머신** | 큰 숫자가 세로로 굴러가는 효과 (자릿수별 translateY) | ✅ |
| **파티클 / 입자 이동** | 줄어드는 영역에서 자라는 영역으로 픽셀이 흐르는 애니메이션 | ⏳ Phase 후속 |

## 3. 시리즈별 상세

### 시리즈 1. 포켓몬 타입의 진화

**기본 정보**
- 시간 축: 1세대 ~ 9세대 (1996 ~ 현재)
- 카테고리: 18개 타입 (노말, 불꽃, 물, 전기, ...)
- 측정값: 각 세대 시점 **누적 마릿수의 타입별 비율(%)**

**📌 확정 사항 (v0.3)**
- **카운팅 방식**: 누적 (그 세대까지 출시된 모든 포켓몬)
- **표시 방식**: 비율(%) 메인. 각 영역 하단에 절대 수치를 작은 텍스트로 함께 표시 (예: "불꽃 12.4% / 87마리")
- **듀얼 타입 처리**: **양쪽 타입에 1씩 중복 카운팅** (시각적으로 더 풍부) ✅ 확정
- **포함**: 일반 포켓몬 + 리전폼 (리전폼은 사실상 다른 포켓몬으로 취급, 타입도 다름)
- **제외**: 메가진화, 다이맥스/거다이맥스 (일시 변신 형태)

**핵심 스토리 포인트**
- 2세대: 악/강철 타입 추가
- 6세대: 페어리 타입 추가 (드래곤 견제용)
- 세대마다 신규 타입 포켓몬이 어떻게 분포되는지

**데이터 소스**
- 1차: A님이 이미 보유한 포켓몬 데이터 (Pokenova)
- 2차: PokéAPI (https://pokeapi.co) — 백업/검증용
- 리전폼 데이터는 별도 처리 필요 (예: 알로라 나인테일 = 얼음/페어리)

**난이도** ★★ (데이터 보유, 카테고리 명확하나 리전폼 정제 필요)

**우선순위** 1순위 (MVP)

### 시리즈 2. 대한민국 국회의 변천

**기본 정보**
- 시간 축: 제헌국회(1948) ~ 22대 국회(현재), **연도별**
- 카테고리: **상위 4개 정당 + 기타 = 총 5개 슬롯** (시기마다 슬롯 주인 변동)
- 측정값: **지역구 의원 의석 수만** (비례대표 제외)

**📌 확정 사항 (v0.3)**
- **5개 슬롯 룰**: 매 시점 의석 수 TOP 4 정당 + 기타(소수정당·무소속 합산). 시기마다 슬롯 주인이 바뀜
- **비례대표 제외**: 위성정당(더불어시민당, 미래한국당 등)도 자동으로 제외
- **정당 이동**: 의원이 당적 변경한 그 시점 기준으로 즉시 카운팅 변경
- **합당/분당 처리**: 합당이면 합치고, 분당이면 나눔 (자연스러운 흐름)
- **시간 단위**: 연도별 (재보궐선거·정당 이동 반영 가능)

**🎨 색상 규칙 (핵심)**
- **기본 색**: 정당 고유색 (이름이 바뀌면 색도 바뀜)
- **계보 테두리**:
  - **2012년 이후** 보수 계보 정당 = **빨간색 테두리** (새누리당부터 보수=빨강 정착)
  - **2012년 이후** 진보 계보 정당 = **파란색 테두리**
  - **2012년 이전** 모든 정당 = 테두리 없음 (당시엔 색 코드 정착 전)
- 효과: 정당명/색이 바뀌어도 "어느 계보인지" 한눈에 보임
- 기타 슬롯은 항상 회색 계열, 테두리 없음

**핵심 스토리 포인트**
- 1960~70년대 양당제 형성
- 3당 합당(1990) → 신한국당 → 한나라당 → 새누리당 → 국민의힘 (보수 계보)
- 평민당 → 새정치국민회의 → 새천년민주당 → 더불어민주당 (진보 계보)
- 2012: 새누리당부터 보수=빨강, 민주당=파랑 색 코드 정착
- 2024 22대 총선: 조국혁신당 등 신당 부상

**데이터 소스 (다중 소스, 우선순위 순)**
1. **공공데이터포털 - 국회사무처_역대 국회 정당별 국회의원 지역분포 API**
   - URL: data.go.kr/data/15126014/openapi.do
   - 가장 적합. 역대 정당별 의원 지역 분포 직접 제공
2. **열린국회정보 - 헌정회 역대 국회의원 API**
   - URL: open.assembly.go.kr
   - 학력/경력/소속정당 정보 포함
3. **중앙선거관리위원회 개방포털**
   - URL: data.nec.go.kr
   - 역대 선거통계 (보조)
4. **popong.com API** (비공식, R&D용)
   - URL: data.popong.com
   - 정당 이동 등 가공된 데이터 제공

**핵심 작업: 정당 색상 매핑 테이블 + 계보 분류**
- 모든 역대 정당의 (정당명, 고유색, 계보=보수/진보/제3당/기타) 매핑 필요
- 보수 계보: 자유당·민주공화당·민주정의당·민자당·신한국당·한나라당·새누리당·자유한국당·미래통합당·국민의힘
- 진보 계보: 평민당·민주당·새정치국민회의·새천년민주당·열린우리당·통합민주당·민주통합당·새정치민주연합·더불어민주당
- 정당색은 각 정당 위키/공식 자료 참조

**난이도** ★★★★ (정당 색상/계보 매핑이 별도 정리 작업급)

**우선순위** 2순위

### 시리즈 7. 한국 PC방 게임 점유율 변천사 ⭐

**기본 정보**
- 시간 축: **월별** (2000년대 초반 ~ 현재)
- 카테고리: **TOP 10 게임** (시점에 따라 달라짐)
- 측정값: PC방 사용 시간 점유율 (%)

**📌 확정 사항 (v0.3)**
- **시간 단위**: 월별 (게임트릭스는 주간 발표 → 월별로 가공: 마지막주 기준 또는 4주 평균)
- **TOP 10**: 매월 점유율 상위 10개 게임 + "기타" 슬롯
- **데이터 신뢰도**: 신경 쓰지 않음. 트렌드/흐름 표현이 목적
- **수집 방식**: **B + C 조합** (아래 참조)

**🛠 데이터 수집 방식 (B+C 조합)**

**[B] 게임트릭스 뉴스 페이지 (메인, 자동 수집)**
- gametrics.com/news 의 매주 발행되는 "주간 종합 게임 동향" 보도자료 수집
- 텍스트 형식으로 TOP 10 + 점유율 명시되어 있음
- 보도자료는 공유 의도라 스크래핑 우려 적음
- **자동화**: PM2 cron으로 매주 일요일 새벽 실행 → 새 보도자료 fetch → TOP 10 추출 → DB 저장
- 월별 가공: 그 달의 마지막 주 데이터 사용

**[C] 나무위키 + 케이벤치 등 IT 매체 (보조, historical 보강)**
- B로 커버 안 되는 과거 시점은 C로 보강
- 나무위키 "PC방/점유율" 문서: 역대 순위가 시대별로 정리되어 있음
- 케이벤치 등 매체의 매월 PC방 순위 보도자료 검색
- 일회성 수동 작업: 2000~게임트릭스 뉴스 아카이브 시작 시점 사이 정제

**활용 원칙**
- 가능한 한 B 데이터 우선 (자동 수집 + 형식 일관성)
- B로 커버 안 되는 과거만 C로 보강 (수동 작업 최소화)
- 두 소스가 겹치는 시점이 있으면 B 데이터 신뢰

**핵심 스토리 포인트 (한국 게임 역사 그 자체)**
- 2000년대 초: 스타크래프트 시대
- 2003~2010: 카트라이더, 던전앤파이터, 서든어택, 메이플스토리, 아이온 격동기
  - 아이온 출시 후 160주 연속 1위 (2008~2011)
- 2012~2017: **롤(LoL) 약 10년 1위 독주**
- 2017.11: **배틀그라운드가 LoL을 제치고 1위** (성인전용 게임 최초)
- 2018~현재: LoL 복귀 → 발로란트·FC온라인·오버워치 2 부상
- 2025.12: **메이플스토리 출시 22년만에 첫 1위** (45.07% 점유율)

**데이터 소스 정리**
1. **게임트릭스 뉴스 페이지** (gametrics.com/news, B 메인) — 주간 종합 동향
2. **나무위키 "PC방/점유율" 문서** (C 보조) — historical
3. **케이벤치 등 IT 매체** (C 보조) — historical 추가 보강
4. **더 로그** (검증용, 비교 가능 시점만)

**용어 정리**
- **PC방 점유율**: 표본 PC방의 게임 사용 시간 비율
- **피카 매니져 / 게토PC방**: PC방 관리 프로그램. 데이터 수집 기반
- **F2P / P2W**: 무료 게임 / 결제 우위 게임

**스토리텔링 강점**

**난이도** ★★★ (B 자동화 구축 + C 일회성 수동 정제)
